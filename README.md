# Network_Agent
An autonomous AI agent for network management

## Requirements
- Python >=3.11,<3.14

# Environment variables
- Copy `.env.example` to `.env` and fill in required values

# Install and setup in root folder

## Install venv and Activate .venv environmenton in Windows
- python -m venv .venv
- .venv\Scripts\Activate.ps1

## Install venv and Activate .venv environmenton in Linux
- python3 -m venv .venv
- source .venv/bin/activate

## Install dependencies 
- pip install .
